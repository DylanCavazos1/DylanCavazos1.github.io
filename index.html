Artifact Summary

Originally, this CLI application used a hash table to manage course data loaded from a CSV file. I enhanced the program by:

  • Implementing a Binary Search Tree (BST) for in-memory course retrieval
  • Integrating PostgreSQL for persistent course data between sessions
  • Adding synchronization options between the BST and the database
  • Including CLI functionality to support full CRUD operations

Software Design and Engineering

The first enhancement for the Software Design and Engineering artifact was the conversion to Python from C++ Course Manager project. Originally, this program loaded course data from a CSV file and utilized a Hash Table to store the courses for retrieval and management. This enhancement is aimed at creating a program that utilizes object-oriented principles in a more maintainable and readable language such as Python.
